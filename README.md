# Beatles One - A Beatles Tribute in Uruguay 🇺🇾

Official website for **Beatles One**, a tribute band based in Montevideo, Uruguay.

This project was developed as a responsive website using HTML, CSS and JavaScript, it was built with a focus on clean structure, usability and content management.

## 🚀 Beatles One Web
[https://beatlesone.com.uy](https://beatles-one.netlify.app/)

--------------------------------------------------------------------------------------------------------------------------

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript 
- Netlify (Deployment)
- Git & GitHub (Version Control)

--------------------------------------------------------------------------------------------------------------------------

##  Main Features

- Responsive design (desktop and mobile)
- Dynamic show schedule connected to Google Sheets
- Contact form with email notification and success feedback message
- Direct WhatsApp booking link from the Shows section
- Image gallery with lightbox
- Embedded YouTube video section
- Language switch (Spanish / English)

--------------------------------------------------------------------------------------------------------------------------

## ⚙️ What I Worked On

- Structuring semantic HTML
- Organizing CSS for maintainability with comments
- Testing responsive behavior across screen sizes
- Fixing layout and rendering issues (hero video, text spacing, overlays)
- Managing version control and deployment workflow
